# Data-request templates — official entry/re-entry data

The one thing that would unlock **re-entry** events for exact net P&L is the organizer's
**entry-level data**: the complete list of entrants and, per player, how many bullets they
fired. That data provably exists (it's how the prize pool is reconciled) but isn't published.
Only a real person can request it — below are ready-to-send templates. Replace `[...]`.

---

## A. Tournament organizer (WSOP / Triton / PokerGO–PGT)

**To:**
- WSOP: media@wsop.com (or press contact on wsop.com)
- Triton: the media/press contact on tritonpokerseries.com (Mediarex / GPI media kit)
- PokerGO / PGT: support@pokergo.com or the PGT media contact

**Subject:** Data request — complete entry & re-entry lists for a high-stakes tournament study

Hello [Name / Media Team],

I'm building a **non-commercial research project** analysing profit & loss among the
high-stakes tournament cohort (buy-ins $11,000+). I can compute results precisely for
freezeouts, but for **re-entry** events I'm missing two pieces that only your records hold:

1. the **complete list of unique entrants** for an event (not only the in-the-money finishers), and
2. the **number of entries (bullets) per player**.

Would you be able to share, or point me to, this entry-level data for the following event(s)?

- [Event name, series, date, e.g. "2024 WSOP Event #55 — $250,000 Super High Roller"]
- [Event name, series, date]

Any format is fine (CSV, PDF, a registration export, or even a screenshot of the entries
report). This is for research only; I'm happy to **credit [Organizer] as the data source**,
share the methodology, and respect any restrictions you set. Even an aggregate
"unique entrants vs total entries with the per-player breakdown" would be enormously helpful.

Thank you for considering it,
[Your name]
[Email / project link, e.g. https://github.com/sooflee/voltorb]

### What to ask for, precisely
For each target event you want includable, you need a table of: **player → number of entries
(bullets)**, plus the complete entrant list. With that + the public payouts, the event becomes
an exact net-P&L include. Send me whatever you get back and I'll ingest it.
