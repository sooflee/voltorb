High Roller Ledger

Who in the high-stakes professional cohort is actually winning or losing — live freezeout tournaments, $7,000+ buy-ins, with every buy-in counted (busts included). Data spans 2012–present; the leaderboard defaults to 2020 onward.

⏸ Project on hiatus
This project is incomplete and paused. Net P&L here is winnings minus every buy-in (busts included), which needs a complete entrant list — so only freezeouts with a fully published field are counted. I haven't been able to get the data to go further: full rosters and per-player re-entry counts for re-entry events aren't published anywhere I can reach. The events shown are exact; backing/staking is ignored and all amounts are USD. Everything left out is listed, with reasons, in Coverage.
